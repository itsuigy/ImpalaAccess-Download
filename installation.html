¿Cómo instalar Control de Accessos Impala?


Pasos de instalación para dispositivos Android:


  • Descargar la aplicación:

    • La aplicación de Control de Accesos de Impala está disponible para dispositivos Android. Para descargar la aplicación, se debe de dar clic en el siguiente botón dentro del apartado Inicio de la página web.

  • Da clic en la aplicación con formato .apk:

    • Una vez dado clic en el botón, se descargará un archivo con formato .apk. Este archivo es el que se debe de instalar en el dispositivo Android. Daremos clic en el archivo descargado para empezar con la instalación.

    • Texto

  • Entrar a la aplicación:

    • Una vez que la aplicación fue instalado con éxito, ingresamos a la aplicación para empezar a configurarla. El estilo de la aplicación es como se muestra a continuación en la imagen:

  • Entrar a los ajustes de la aplicación:

    • Una vez dentro de la aplicación, se debe de dar clic en el botón de Ajustes que se encuentra en la parte superior derecha de la pantalla.

  • Cambiar los ajustes de la aplicación:

    • Dentro de los ajustes, se pueden cambiar los ajustes de la aplicación, como el tiempo de alerta, el sonido de alerta, las probabilidades de los accesos, entre otros.

  • Listo para usar:

    • Una vez realizado las configuraciones, deberás escribir la contraseña de verificación para guardar los cambios y listo, la aplicación está lista para usarse.

Punto a tomar en cuenta:


  • La aplicación de Control de Accesos de Impala está disponible para dispositivos Android:

    • La aplicación se encuentra actualmente desarrollada únicamente para dispositivos Android. No se encuentra disponible para dispositivos iOS u otros sistemas operativos.
